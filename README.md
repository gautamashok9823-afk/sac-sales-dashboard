# SAC Sales Dashboard

This repository is used for version control and backup of SAP Analytics Cloud (SAC) assets.

## Folder Structure

- exports/story   -> SAC Story exports (ZIP)
- exports/model   -> SAC Model exports (ZIP)
- documentation   -> KPI definitions & business logic
- screenshots     -> Dashboard screenshots
- sql             -> SQL / HANA views

## Usage

1. Export story/model from SAC
2. Upload ZIP here
3. Commit with version comment
4. Use for backup or transport
